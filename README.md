# Tutor Connect Cameroon

A cloud-based platform connecting students in Cameroon with qualified private tutors. Built with modern cloud technologies for scalability, security, and ease of deployment.

## Features
- User authentication (students & tutors)
- Tutor search and filtering
- Booking system
- Tutor profile setup
- Dashboard for both users
- Responsive mobile-first design
- Cloud Firestore database
- Firebase Authentication
- Firebase Hosting ready

## Tech Stack
- React.js
- Firebase
- Cloud Firestore
- Firebase Auth
- Firebase Hosting
